# SOPT 합동세미나
## SOPT-COMICS-SERVER

### assignment
SOPT 웹툰을 제작한다.

### role
Client : 신승민, 김정음, 양승희
Server : 윤희성, 석영현

### frame work
Client : Android (kotlin)
Server : NodeJS, mysql

### use library
#### Server 
'express' : express 모듈

'crypto' : 암호화 모듈

'promise-mysql' : mysql 접속 모듈

'multer' : multipart/form-data 모듈

'aws-sdk', 'multer-s3' : 아마존 s3 접속 모듈
